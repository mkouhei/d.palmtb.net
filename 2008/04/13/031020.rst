aliasディレクティブとDirectoryディレクティブで指定すると、動いてくれない。例えばこんな風に↓。

.. code-block:: none

   NameVirtualHost *
   VirtualHost *
   (snip)
   	DocumentRoot /var/www/
   	Directory /
   		Options None
   		AllowOverride None
   	/Directory
   (snip)
   
   	SuexecUserGroup hoge hoge
           Alias /rails/ /var/www/rails/public/
           Directory /var/www/rails/public/
                   Options ExecCGI FollowSymlinks
                   AllowOverride all
                   Allow from all
                   Order allow,deny
           /Directory
   (snip)
   /VirtualHost


VirtualHostを別にしてDocumentRootを指定してやるとちゃんと動く。

.. code-block:: none

   VirtualHost rails.hoge.com
   	DocumentRoot /var/www/rails/public
   
   	SuexecUserGroup hoge hoge
   
           Alias /rails/ /var/www/rails/public/
   	Directory /var/www/rails/public/
   		Options ExecCGI FollowSymlinks
   		AllowOverride all
   		Allow from all
   		Order allow,deny
   	/Directory
   (snip)
   /VirtualHost
